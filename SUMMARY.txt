================================================================================
                     TROUBLESHOOTING
================================================================================

App won't launch?
  → Verify: PyQt6 installed (pip install PyQt6)
  → Check: Python 3.8+ version

Defense not working?
  → Check: "Enable Defense" button clicked
  → Check: Defense level not PASSIVE
  → Check: Console for error messages

Features missing?
  → Check: Module installed and available
  → Check: No import errors in console
  → See: LAUNCH_INSTRUCTIONS.md
